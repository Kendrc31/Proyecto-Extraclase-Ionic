# Proyecto Extraclase - Gestión de Locales (Ionic + React)

Este proyecto es una aplicación móvil desarrollada con Ionic Framework, React y Capacitor.

## Requerimientos cumplidos:
- **CRUD Completo:** Agregar, Editar, Eliminar y Limpiar.
- **Búsqueda:** Filtro en tiempo real por descripción.
- **Validaciones:** Campos obligatorios y descripción como llave primaria única.
- **Navegación:** Uso de Tabs (Lista de locales y Formulario de gestión).

## Instrucciones para ejecutar:
1. Descargar o clonar el repositorio.
2. Ejecutar `npm install` para instalar las dependencias.
3. Ejecutar `ionic build` para compilar la parte web.
4. Ejecutar `npx cap sync` para sincronizar con Android.
5. Abrir la carpeta `android` en Android Studio y ejecutar en un emulador o dispositivo físico.
